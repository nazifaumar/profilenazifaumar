PROFILE

Hallo semuanya!

Perkenalkan nama saya Nazifa Umar, kalian bisa panggil saya Ziffa, saya berumur 16 tahun, saya tinggal di Bogor dan lahir pada tanggal 16 November 2005. Saya adalah siswa kelas 10 yang bersekolah di SMK Wikrama Bogor dan mengambil jurusan Pengembangan Perangkat Lunak dan GIM (PPLG).

Salah satu hobi saya adalah membaca, membaca adalah kegiatan yang menarik bagi saya, saya suka membaca novel atau Wattpad. Selain membaca buku, saya juga suka menyanyi dan menonton. Saya selalu menyanyikan lagu ketika saya mandi, itu adalah hal yang menarik bagi saya dan saya bisa merasa sangat santai, tetapi jika teman-teman saya meminta saya untuk bernyanyi di atas panggung, saya merasa sangat takut hehehe. Ambisi saya adalah belajar dan bekerja keras sehingga saya bisa mencapai impian saya untuk menjadi seorang programmer.

Sekian dari saya, terima kasih atas perhatiannya ^-^ !
